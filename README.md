# Iron Dash 3000

**Iron Dash 3000** is a lightweight, dependency-free endless runner game built using Vanilla JavaScript and the HTML5 Canvas API.

The project mimics the mechanics of the classic Chrome Dino game but features procedurally generated graphics representing the Marvel Cinematic Universe characters Iron Man and Thanos. It emphasizes performance, portability, and zero-asset deployment.

### Overview

Iron Dash 3000 was designed as a single-file application.
All visual elements, including the player character (Iron Man), obstacles (Thanos), and collectibles (Infinity Stones) are rendered in real-time using JavaScript vector drawing commands.

The specific nomenclature "3000" references the iconic quote from *Avengers: Endgame*, serving as a thematic nod to the franchise.

Since this is a client-side static file, no server environment (Node.js, Python, etc.) is required to run the application.
```
Download the `index.html` file.
Open the file directly in any modern web browser (Chrome, Firefox, Safari, Edge).
```
